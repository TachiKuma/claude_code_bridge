# Iteration 001

## Current Understanding

本 goal 已有批准的 design/review/checklist，终态要求不是简单让 Rmux 路径能跑，而是把 project namespace lifecycle 的 authority 从 legacy tmux socket/session 收束到 canonical `namespace_ref` 和 backend capability。

本轮恢复时发现工作区已有一批相关未提交改动：namespace state/event 已部分支持 canonical 字段，`namespace_projection.py` 已存在，`project_focus/tmux.py`、`slot_replacement.py` 等也已有局部改动。按 dirty worktree 规则，本轮没有回退这些改动，而是在其基础上继续。

## Implementation Approach

方案深度判断：该路径属于长期维护的 backend authority 边界，不能用 CLI 拼接或 alias-only 方案降级。本轮采用真实 capability 接线：

- Rmux foreground attach 下沉到 `RmuxBackend.attach_namespace()`，由 backend command client 负责 foreground stdio。
- `start_foreground.py` 的 Rmux 分支只读取 canonical payload 并调用 backend attach capability，不再拼 `rmux attach-session` 命令。
- `ProjectNamespaceController.root_pane_id()`、destroy 和 backend factory 从 `namespace_ref` 构造 backend，避免 Rmux authority 回退到 `tmux_socket_path`。
- Rmux destroy/recreate 只 kill project namespace，不默认 kill shared backend server/daemon。

## Changes This Iteration

- `lib/terminal_runtime/rmux_backend_runtime/namespace.py`：实现 `attach_namespace()`，使用 `attach-session` foreground command client，并保留 `MuxCommandError` 映射。
- `lib/cli/services/start_foreground.py`：Rmux attach 改为 backend capability；payload helper 支持嵌套 `namespace_ref`；移除 CLI 层 direct Rmux command builder。
- `lib/ccbd/services/project_namespace_runtime/backend.py`、`controller.py`、`namespace_projection.py`：backend factory 支持 canonical `namespace_ref`，并对 path-like `ipc_ref` 才作为 socket path。
- `lib/ccbd/services/project_namespace_runtime/destroy.py`、`ensure_state.py`、`records.py`：created/destroyed state/event 写入 canonical backend/ipc 字段；非 tmux mux backend recreate/destroy 按 session kill namespace。
- `lib/ccbd/services/project_namespace_state_runtime/models.py`：Rmux/psmux restore 不再把 `namespace_ref.ipc_ref` 回填成 legacy `tmux_socket_path`。
- `lib/terminal_runtime/fake_mux_backend.py`：补 public `namespace_ref()`，让 fake backend 满足 mux contract。
- 测试更新覆盖 Rmux attach capability、canonical backend factory、Rmux namespace destroy 和 state restore。
